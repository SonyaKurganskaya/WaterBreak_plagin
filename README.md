# Water Break - Для напоминания о необходимости выпить воды
## Автор
Курганская София Александровна М3122
## Описание
Как часто Вы пьете воду во время работы? Плагин Water Break поможет поддерживать здоровый образ жизни, даже когда у Вас горят дедлайны. Каждые 20 минут повявляются уведомления о "водных" перерывах, которые помогут немного принести пользу Вашему здоровью! ❤️

## Возможности 
### <span style="color: pink;">Автоматические напоминания</span> 
  * *уведомления обновляются каждые 20 минут*
  * *уведомления не отвлекают от рабочего процесса*
  
### <span style="color: pink;">Простое управление</span> 
  * *можно отключить уведомления в одно нажатие*
  * *можно включить уведомления в одно нажатие*
  
### <span style="color: pink;">Стабильность</span> 
  * *не влияет на работу VS Code *
  * *использование ресурсов - минимально*
  
## Использование 
1. Откройте необходимый проект в VS Code 
2. Нажмите на кнопку **waterbreak** в правом нижнем углу
3. Пользуйтесь плагином и не забывайте выпивать суточную норму воды!
